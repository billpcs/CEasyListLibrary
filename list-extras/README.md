list-extras
===================

A Library for easy List Handling.

Only **ints** are supported. This means that you can *not* have list of list , list of floats etc.

Methods that are implemented:

 + **append()**
 
 + **insert()**
 
 + **removeitem()**
 
 + **pop()**
 
 + **index()**
 
 + **count()**
 
 + **sort()**
 
 + **reverse()**

 + **printlist()**

 + **filllist()**

 + **setlist()**

 + **length()**

 + **getitem()**
